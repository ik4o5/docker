This is a short introduction to the anaconda environment that was used to develop
the export functionality.

- open skelleton.yml file in this directory and change the "name" to whatever you
  would like to call your environment
- use the command "conda env create -f skeleton.yml"
- run the command "pip install -U -e ego.io\"
- run the command "pip install -U -e ding0\"
- run the command "pip install -U -e eDisGo\"
